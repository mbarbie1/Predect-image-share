Features on the roadmap

  3. Image analysis algorithms
  4. Download derived (image analysis) image stacks exported as multi-page tif-file
  5. Image viewer
    1. Gray and RGB 3D stacks and 2D images
    2. Zoom
      1. Gray and RGB 3D stacks and 2D images
      2. Zoom

        • Zoom factor
      3. Filters for brightness, contrast using min. and max. pixel intensity
      4. Channel slider
      5. Z-layer slider
